class celery.utils.term.colored(*s, **kwargs)¶

Terminal colored text.

Example::
>>> c = colored(enabled=True)
>>> print(str(c.red("the quick "), c.blue("brown ", c.bold("fox ")),
...       c.magenta(c.underline("jumps over")),
...       c.yellow(" the lazy "),
...       c.green("dog ")))
