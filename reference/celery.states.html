Documentation has moved

Celery is now using Read the Docs to host the documentation for the development version, where the pages are automatically updated as new changes are made (YAY!)

The new location for this page is:
http://docs.celeryproject.org/en/master/reference/celery.states.html.

If you wanted documentation for the latest stable version instead, please go to:
http://docs.celeryproject.org/en/latest/reference/celery.states.html.

PDF, Epub, and HTMLZip documents for every Celery version, including development -- is located here: https://readthedocs.org/projects/celery/downloads/
